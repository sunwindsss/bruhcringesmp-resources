https\://raw.githubusercontent.com/sunwindsss/bruhcringesmp-resources/main/bruhcringesmp-resourcepack.zip

sha1 file checksum. use raw link to download resource .zip, and put it in this site to check sha1 checksum:
https://emn178.github.io/online-tools/sha1_checksum.html

pack format. ensure the resource pack follows correct minecraft version pack format:
https://minecraft.wiki/w/Pack_format#List_of_pack_formats
(for version 26.1.2 it is 84.0)
